# 《平台系统说明》Word 大纲（不写研究背景）

1. 系统范围与交付物
2. 总体架构
   - 离线构建链路图
   - 在线问答链路图
3. 数据源与本地根文件
   - DiaKG 根文件、目录、校验值、许可/使用条款
   - RetinaMNIST 根文件、目录、校验值、许可
   - 人工别名字典
4. 知识模型
   - 实体类型表
   - 关系类型表和方向
   - 节点/边属性
   - 多模态图像建模
5. 平台模块与部件
   - 数据采集与解析
   - 实体规范化和关系映射
   - 质量检查
   - 图谱存储与可视化
   - 问题理解、实体链接、查询规划
   - 证据与答案组织
   - 图像检索/可选图像模型
   - API 与前端
6. 数据链路、输入输出与方法
   - 每个模块的输入、输出、关键算法、失败处理
7. 部署与复现
   - 环境、依赖、Docker、命令、固定随机种子

8. 质量与评估
   - 图谱规模与分布
   - 重复率、孤立节点率、schema 合规率、证据覆盖率
   - 问答用例正确率和延迟
   - 可选图像模型指标
9. 复现案例（至少 3–5 个）

   - Case 3：糖尿病视网膜病变相关检查和受累部位
   - Case 4：按分级展示眼底图像并返回图谱解释
   - Case 5（加分）：上传固定测试图，返回实验性分级、相似图和 KG 证据
10. 已知限制与安全声明
11. 附录
   - API 示例
   - 目录结构
   - 版本信息
   - 数据引用
